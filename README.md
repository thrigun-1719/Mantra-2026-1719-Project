🌍 BHARAT YATRA — Project README

📌 What is this project?

This is a complete travel website built as a frontend development assignment for MANTRA 2026 Summer School. It's a fully functional, responsive website showcasing international destinations, tour packages, booking forms, and more.

🧩 Pages Included

Page	                  File	                                   What it contains

Home	                index.html	    Hero banner, stats, featured destinations, packages preview, quick booking form 

About	                about.html	                  Company story, learning outcomes, mission

Destinations	        destinations.html	          10 international destination cards with details and tags

Packages	            packages.html	              8 tour packages with prices + cost calculator

Gallery	                gallery.html	              12 gallery items with filter buttons (All, City, Beach, etc.)

Booking  	            booking.html	              Full booking form with JavaScript validation

Contact	                `contact.html**               | Contact details, office hours, social links, contact form |
